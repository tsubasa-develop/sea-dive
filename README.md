# Sea Dive — 蒼の図鑑

three.js 製の海中探索・生物撮影ゲーム。一人称のダイバーとなって海を自由に泳ぎ、
生き物を撮影して図鑑を完成させることが目的です。外部3Dモデル・画像素材は一切使わず、
地形・サンゴ・魚・エフェクトのすべてをコードで手続き生成しています。

## セットアップ

```bash
npm install
npm run dev     # 開発サーバー (http://localhost:5173)
npm run build   # 型チェック + プロダクションビルド
```

## ゲームの流れ

1. **サンゴ礁(〜30m)** からスタート。まずは身近な魚を撮影して図鑑に登録する
2. 図鑑 **6種** で可潜深度が **90m** に解放 → 断崖(ドロップオフ)と沈没船エリアへ
3. 図鑑 **14種** で **深度無制限** に解放 → 熱水噴出孔・鯨骨・海底洞窟のある深海へ

### 図鑑と希少度

全 **27種**。それぞれ現実の生態を反映した行動をとります。

- ★1〜2 … よく見かける(ルリスズメダイ、イワシの群れ など)
- ★3 … 少し珍しい(アオウミガメ、ミナミハンドウイルカ、バラクーダ など)
- ★4 … 珍しい(擬態するワモンダコ、チョウチンアンコウ、マンタ など)
- ★5 … 伝説級。**ジンベエザメ / ザトウクジラ / リュウグウノツカイ / シーラカンス**。
  回遊種は低確率の「遭遇イベント」でしか現れません。予兆のテキストと音に注意。

生態の例:

- カクレクマノミはイソギンチャクから離れず、驚くと巣に逃げ込む
- アオウミガメやナポレオンフィッシュは、静かに待つダイバーに近寄ってくる
- ワモンダコは近づくと岩の色に擬態する
- シーラカンスは深海の洞窟のそばから離れない

### 写真の採点

シャッターを切ると、構図(中央度)・被写体の大きさ・距離・向き・希少度・
群れ / 多種の共演ボーナスからスコアとランク(S/A/B/C)が算出されます。
写真はブラウザ内(IndexedDB)に保存され、アルバムからJPEGとしてダウンロードできます。

## デバッグモード

タイトル画面の右下にある小さな **dev** を押すと、デバッグモードで開始します。

- 遭遇イベント(クジラ・ジンベエザメ等)が高頻度でサイクル出現
- 可潜深度が最初から無制限
- `1 / 2 / 3` … サンゴ礁 / ドロップオフ / 深海へテレポート
- `E` … イベント種を即時召喚(押すたびに次の種へ)

## 技術構成

- **three.js + TypeScript + Vite**
- 地形: ノイズ関数による解析的ハイトフィールド(衝突・遮蔽判定も同じ関数を共有)
- 魚: パラメトリックな体型生成 + Canvasテクスチャ + 頂点シェーダーの遊泳アニメーション
  (群れは InstancedMesh + 位相attribute)
- 群れAI: Boids(分離・整列・結合)+ 生息域アンカー + プレイヤー逃避
- 環境表現: 深度連動のフォグ / 太陽光減衰(深いほど青く)、コースティクス、ゴッドレイ、
  頂点変位によるうねる海面(波頭・さざ波・スネルの窓)、マリンスノー、発光生物のエミッシブ、
  PMREM環境マップ、UnrealBloomによる淡い発光
- サウンド: WebAudioでの手続き生成(環境音・呼気・シャッター・発見ジングル)
- 進行状況: localStorage、写真: IndexedDB
