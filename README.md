# Sea Dive — 蒼の図鑑

three.js 製の海中探索・生物撮影ゲーム。一人称のダイバーとなって海を自由に泳ぎ、
生き物を撮影して図鑑を完成させることが目的です。外部3Dモデル・画像素材は一切使わず、
地形・サンゴ・魚・エフェクトのすべてをコードで手続き生成しています。

## セットアップ

```bash
npm install
npm run dev     # 開発サーバー (http://localhost:5173)
npm run build   # 型チェック + プロダクションビルド
```

## ゲームの流れ

**フリー探索モード**: 最初からすべての深度を自由に潜れます。
サンゴ礁 → 断崖(ドロップオフ) → 深海と潜りながら、生き物を撮影して図鑑を完成させましょう。

地形は**起動のたびにランダム生成**されます。リング状の構造(礁 → 断崖 → 深海)は
保たれますが、海山・海底谷・沈没船や洞窟の位置は毎回変わります。

### 神秘の遺構

海のどこかに、誰が作ったのかわからない構造物が沈んでいます。

- **海底遺跡** … 崩れた石柱の環。淡い光が漂う
- **モノリス** … 深海に立つ黒曜石の板。青いルーンが刻まれている
- **石の門** … 海底谷の入り口にそびえる巨大な門
- **光るクリスタル群** … 深海の谷底で青く輝く

### ⚠ サメの襲撃

**深海(78m超)や沖合に長居すると、低確率でホホジロザメに狙われます。**
警告のテキストと低い音が鳴ったら、**浅場(35mより浅く)へ上がるか、礁の内側へ**逃げ込むこと。
どちらかを満たせばサメは追ってきません。

旋回しながら距離を詰め、突進してきます。**1回噛まれると重傷**(画面が赤く脈打つ)、
**2回噛まれると死亡**し、新しい海で再スタートになります(写真と図鑑は失われません)。
サメをやり過ごして生き延びれば、傷は癒えます。
あえて撮影すれば図鑑最高クラス(★5)の獲物ですが、命がけです。

### 図鑑と希少度

全 **61種**。それぞれ現実の生態を反映した行動をとります。

- ★1〜2 … よく見かける(ルリスズメダイ、アカネハナゴイ、イワシの群れ など)
- ★3 … 少し珍しい(アオウミガメ、チンアナゴ、ネムリブカ、キハダマグロ など)
- ★4 … 珍しい(擬態するワモンダコ、チョウチンアンコウ、マンタ、イタチザメ など)
- ★5 … 伝説級。**ジンベエザメ / ザトウクジラ / シロナガスクジラ / リュウグウノツカイ /
  シーラカンス / ミツクリザメ / ラブカ / ダイオウイカ / ホホジロザメ**。
  回遊種は低確率の「遭遇イベント」でしか現れません。予兆のテキストと音に注意。
- そして深淵には、既知のどの生物とも一致しない**全長40mの巨大魚影**が目撃されている…

生態の例:

- カクレクマノミはイソギンチャクから離れず、驚くと巣に逃げ込む
- アオウミガメやナポレオンフィッシュは、静かに待つダイバーに近寄ってくる
- チンアナゴは近づくと砂に引っ込む
- ワモンダコは近づくと岩の色に擬態する
- シーラカンスは深海の洞窟のそばから離れない

### 写真の採点

シャッターを切ると、構図(中央度)・被写体の大きさ・距離・向き・希少度・
群れ / 多種の共演ボーナスからスコアとランク(S/A/B/C)が算出されます。
写真はブラウザ内(IndexedDB)に保存され、アルバムからJPEGとしてダウンロードできます。

## 3Dモデルの導入(マーケットプレイス活用)

魚や生物は手続き生成がデフォルトですが、**GLBモデルを置くだけで実モデルに自動置き換え**できます。

1. [Sketchfab](https://sketchfab.com)(CCライセンス作品は glTF/GLB でDL可)や CGTrader 等でモデルを入手
2. `public/models/` に `.glb` を配置
3. `public/models/manifest.json` に種IDを登録:

```json
{
  "great_white": { "file": "great_white.glb", "yaw": 3.1416 },
  "green_turtle": { "file": "turtle.glb", "scale": 1.1 },
  "manta":       { "file": "manta.glb", "noSwim": true }
}
```

| 項目 | 説明 |
| --- | --- |
| `file` | `public/models/` からの相対パス(必須) |
| `yaw` / `pitch` / `roll` | 向き補正(ラジアン)。ゲームは **+Z が前方** |
| `scale` | 体長正規化後の追加倍率(省略時 1) |
| `noSwim` | 骨なしモデルへの泳ぎシェーダー適用を止める |
| `swimAxis` | 泳ぎの体軸(`z` / `-z` / `x` / `-x`、省略時 `z`) |

- スケールは種の体長(SpeciesData の `length`)へ**自動正規化**されるので、モデルの元サイズは不問
- **スキンアニメ付きモデルは最初のクリップを自動再生**。骨なしモデルには手続き泳ぎを注入
- Draco圧縮・KTX2テクスチャ対応(デコーダー同梱)。読み込み失敗時は手続き生成にフォールバック
- 種IDは `src/creatures/SpeciesData.ts` を参照(全61種)
- 群れ(イワシ等の school 種)はインスタンス描画のため対象外

**ライセンス注意**: CC-BY はクレジット表記が必要。Unity Asset Store のアセットは原則 Unity 専用です。

## 技術構成

- **three.js + TypeScript + Vite**
- 地形: シード付きノイズによる解析的ハイトフィールドのランダム生成
  (衝突・遮蔽判定も同じ関数を共有)。海山・海底谷もシードから決定
- 魚: パラメトリックな体型生成(背腹非対称プロファイル + 曲線輪郭のひれ)
  + Canvasテクスチャ(鱗・エラ・側線・虹彩・鰭条・虹色光沢) + バンプマップ
  + 頂点シェーダーの遊泳アニメーション(群れは InstancedMesh + 位相attribute)
- サンゴ礁: 段積みテーブルサンゴ・樹状サンゴ・シャコガイ・樽カイメンなど十数種を
  数千インスタンス配置(タイプごとに1ドローコール)
- 衝突: 地形は解析関数の多点サンプリング、大型オブジェクトは球コライダー(約1,100個)で
  カメラのめり込みを防止
- 群れAI: Boids(分離・整列・結合)+ 生息域アンカー + プレイヤー逃避
- 環境表現: 深度連動のフォグ / 太陽光減衰(深いほど青く)、コースティクス、ゴッドレイ、
  頂点変位によるうねる海面(波頭・さざ波・スネルの窓)、マリンスノー、発光生物のエミッシブ、
  PMREM環境マップ
- ポストプロセス: GTAO(環境遮蔽) + UnrealBloom + MSAA(4x)
- 質感: Poly Haven の実写テクスチャを地形(砂は2周波ブレンド、崖・岩はトライプラナー投影)に適用
- サウンド: WebAudioでの手続き生成(環境音・呼気・シャッター・発見ジングル)
- 進行状況: localStorage、写真: IndexedDB
